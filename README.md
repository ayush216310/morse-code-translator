A basic GUI which helps convert morse-code to string and string to morse-code. Select out of the two options to perform the translation.
Instructions:
1. Only the input window will be open, the other window will be locked.
2. To input morse-code, use the spacebar, shift and enter button.
3. Single press the spacebar to type a dot symbol
4. Long press the spacebar to type a dash symbol
5. Once you have typed the morse-code for a single letter, press the commit letter button to translate
6. Press the shift button for space, it may first appear as a question mark in the display, but it will rectify itself later
7. Press the enter button once to enter next line, press it twice to end the program

The string to morse-code translation will take place parallely as you type in the string.
This program is meant for learning morse-code. Suggestions are welcome.
Thank you!
